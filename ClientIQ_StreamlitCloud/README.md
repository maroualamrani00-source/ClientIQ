# ClientIQ — Application Web de Gestion Clients

## 📌 Description
ClientIQ est une application web professionnelle de gestion et d'analyse clients, avec intelligence artificielle intégrée pour la personnalisation marketing.

## 🚀 Déploiement sur Streamlit Cloud
1. Créez un dépôt GitHub avec ces fichiers
2. Allez sur [share.streamlit.io](https://share.streamlit.io)
3. Connectez votre compte GitHub
4. Sélectionnez ce dépôt
5. Dans **Secrets**, ajoutez : `ANTHROPIC_API_KEY` = votre clé
6. Cliquez **Deploy**

## 📦 Requirements
- Python 3.9+
- Voir `requirements.txt`
